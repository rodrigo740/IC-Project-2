Detailed Description

Main

The frames should be encoded using spatial predictive coding based on thenon-linear predictor of JPEG-LS or the 7 JPEG linear predictors; Entropy coding should be performed using Golomb codes; All the information required by the decoder should be included in the bit-stream (video format, frame size, encoder parameters, etc.)

The program receives 2 input arguments, the first one being the path to the input image and the second one the path to save the decompressed image.

After this the program splits the input image by its three color components(RGB) and performs the necessary calculations to obtain the YUV values.
